It contains various practice exercises, experiments, and small examples that help me understand how Express works in real-world backend development. I will continue updating this repository as I learn new concepts and build more projects.

## Running the Examples

1. Clone the repository

```bash
git clone https://github.com/ShauryaaSharma/Express-Progress.git
```

2. Navigate into the repository

```bash
cd Express-Progress
```

3. Install dependencies

```bash
npm install
npm install express
```

4. Run the server using nodemon

```bash
npx nodemon index.js
```

Replace `index.js` with the entry file of the example you want to run.

## Testing the APIs

Many of the examples in this repository involve creating and testing APIs. To interact with these APIs easily, it is recommended to use Postman.

Postman allows you to:

- Send HTTP requests such as GET, POST, PUT, and DELETE
- Add headers, authentication tokens, and request bodies
- Test and debug API endpoints
- Inspect server responses in an organized way

After running the Express server, you can send requests to endpoints such as:

```
http://localhost:3000/
http://localhost:3000/users
```
